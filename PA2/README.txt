Para correr o algoritmo, basta carregar, para o SICStus Prolog,
o ficheiro "TV Programation Optimizer.pl".

Neste ficheiro, existe um local onde diz "load_files(dataM)".
"dataM" corresponde ao ficheiro "dataM.pl" que é uma configuração
com complexidade média. Basta alterar o nome para o nome de um dos
outros ficheiros e carregar para o SICStus Prolog, o ficheiro
"TV Programming Optimizer.pl" para essa configuração ficar
disponível.

Para iniciar o algoritmo, basta escrever na linha de comandos
"betterSol.".
